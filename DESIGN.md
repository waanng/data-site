# 数据分析网站设计方案

## 🎨 整体设计理念

### 风格定位
- **主题**：现代数据科学风格
- **色调**：深蓝 + 白色 + 科技绿/橙点缀
- **布局**：简洁、专业、信息密度适中
- **交互**：平滑过渡、悬停效果

### 色彩方案
```
主色：#1a237e (深蓝) - 专业、可信赖
辅助色：#00bcd4 (青色) - 科技感
强调色：#ff9800 (橙色) - 数据可视化常用
背景：#f5f7fa (浅灰) 和 #ffffff (白)
文字：#212529 (深灰) 和 #6c757d (中灰)
```

### 4. Notebook展示页 (ml-analysis/notebook.html)

两种方式：

**方式A：嵌入nbviewer**
- 使用iframe嵌入nbviewer.org的渲染结果
- 优点：加载快，格式完美
- 缺点：依赖外部服务

**方式B：本地nbconvert转换**
- 将.ipynb转换为.html
- 本地托管，完全可控
- 可以自定义样式

#### 设计要点
- 左侧：目录导航（自动提取heading）
- 右侧：Notebook内容
- 顶部：工具栏（字体大小、主题切换、全屏）
- 代码块：语法高亮，可折叠
- 图表：点击放大查看

## 🔧 技术实现

### 核心技术栈
- **HTML5** + **CSS3** (Flexbox/Grid)
- **原生JavaScript** (无需框架，轻量)
- **Font Awesome** - 图标
- **Google Fonts** - 字体 (Inter + JetBrains Mono)

### Notebook展示方案

推荐方案：**本地nbconvert + 自定义样式**

```bash
# 转换命令
jupyter nbconvert --to html --template lab \
  --no-input \
  --output-dir ./data-site/ml-analysis/assets/notebook \
  ml_analysis_notebook_executed.ipynb
```

然后：
1. 将生成的HTML嵌入到页面iframe中
2. 或者提取body内容，注入到自定义容器中
3. 添加左侧目录导航（通过JS解析heading生成）

### 响应式断点
```css
/* 手机 */
@media (max-width: 576px) { ... }

/* 平板 */
@media (min-width: 577px) and (max-width: 991px) { ... }

/* 桌面 */
@media (min-width: 992px) { ... }

/* 大屏 */
@media (min-width: 1400px) { ... }
```

## 📝 内容规划

### 首页文案

**标题**：数据驱动的洞察
**副标题**：探索数据背后的故事，用机器学习发现规律，用可视化传递价值。
**CTA按钮**：查看项目

### 机器学习项目文案

**项目标题**：客户购买行为预测 - 端到端机器学习分析
**简介**：
这是一个完整的二分类机器学习项目，从数据探索、特征工程到模型训练与评估，展示了标准的数据科学工作流程。使用4,050条客户数据，预测客户是否会购买产品。

**关键技术**：
- 数据清洗与探索性分析（EDA）
- 特征工程：标签编码、分箱、交互特征
- 模型对比：逻辑回归、决策树、随机森林
- 模型评估：准确率、精确率、召回率、F1、AUC
- 可视化：9宫格综合分析仪表盘

**项目亮点**：
1. 📊 完整的EDA可视化分析
2. 🤖 3种机器学习模型对比
3. 📈 详细的特征重要性分析
4. 💡 业务洞察与可执行建议

## 🚀 部署配置

### GitHub Pages配置

1. 在仓库设置中启用GitHub Pages
2. 选择部署分支：main
3. 选择文件夹：/ (root) 或 /docs
4. 自定义域名：data.waanng.cn

### DNS配置

在域名服务商处添加CNAME记录：
```
主机记录：data
记录类型：CNAME
记录值：waanng.github.io
```

## 🎯 扩展性考虑

未来添加新项目时，只需：
1. 在projects/下创建新项目文件夹
2. 复制项目模板（index.html + notebook.html）
3. 修改内容，添加项目卡片到首页和项目列表页
4. 更新导航链接

网站架构支持无限扩展，每个项目独立，互不影响。
